driverTxRxData This function implements the data Transmission/Reception on the BUS. The function shall be implemented as a blockant with timeout. In case no data should be received, the function shall return a all 0 frame.

Parameters
frameThis is the frame to be transmitted;
tmoThis is the max time in ms the driver shall wait for a data reception.
Returns
  • The received frame in case of valid reception.
  • All 0's frame in case of no data reception.

openDriverPort This function executes several tasks:

  • Initialize the workerthread at the first call;
  • Initialize the hardware if not initialized
  • Opens a receiver filter

The communication however is not initiated until a runDriver() command is called
