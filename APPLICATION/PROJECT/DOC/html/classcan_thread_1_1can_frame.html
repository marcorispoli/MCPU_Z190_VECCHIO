Detailed Description

This is the class handling the Can Frame buffers.

The class identifies the basic message items coded ionto the data fields of a CAN frame buffer. In particolar the class handle:

  • The nodeId: the device node sending the frame;
  • The CANOpen frame class: this is the CanId (11 bit field len) & 0xFF80. In the CANOpen protocol the only admitted code is 0x580 (SDO frame).
  • Idx: Object dictionary Index;
  • SubIdx: Object Dicionary sub index;
  • frameCode: ReadData type, Write data type o error frame code;
  • Data content: four bytes of packet data content.
