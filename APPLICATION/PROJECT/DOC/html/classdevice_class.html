◆ statusChangedSgn

void deviceClass::statusChangedSgn ( uchar  index)
signal

This signal is generated when a Status register changes its value.

When a decoded can frame relates to a READ-STATUS command, the STATUS register content received from the device is compared with the current value. If the value should change, the signal can be generated.

Parameters
indexthis is the index of the changed Status register
