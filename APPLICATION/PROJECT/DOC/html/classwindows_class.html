openParentWindow This method open the parent window

The parent window is the last closed window. The class takes the history of the closing windows allowing to retrive the last closed one. This is useful in the case where an asynch window show up for a temporary activity then the previous window shall be ripristinated.

Parameters
optBitfield option flags. Available flags are:
  • windowsClass::_WINDOW_NOINIT: prevent to call the windowsClass::initWindow() method for the opening window;
  • windowsClass::_WINDOW_NOEXIT: prevent to call the windowsClass::exitWindow() method for the closing window; Open the parent window if there is one

◆ openWindow()

void _inline windowsClass::openWindow ( int  opt = 0)
inline

openWindow This method open the Window

This method shall be used in order to open the window of this class instance.

When this method is called, the following actions take place:

  • The windowsClass::exitWindow() of the currently open windows is called;
  • The the windowsClass::initWindow() of the opening window is called;

If the window is already open the method do nothing, unless the option flag windowsClass::_WINDOW_FORCE is passed as parameter:
in the latter case, the method windowsClass::initWindow() is called anyway.

Parameters
optBitfield option flags. Available flags are:
  • windowsClass::_WINDOW_FORCE: force an already open window to call the windowsClass::initWindow() method;
  • windowsClass::_WINDOW_NOINIT: prevent to call the windowsClass::initWindow() method;
  • windowsClass::_WINDOW_NOEXIT: prevent to call the windowsClass::exitWindow() method for the closing window;
