Detailed Description

This class handle the whole set of connected clients.

The class handle the queue of the connected class;

The class forwards the received can frame to the Client registered with the address of the Can frame sender;

The class is responsible to erase a disconnected client socket.

slot

Slot to be connected to the canDriver reception signal.

This function receives the data coming from the CAN network.

The Data packet received from the CAN network is forwarded to
the connected Slaves that has been registered with the canId identifier
or with the broadcast address (0).

The Data is put in the socket packet as for the protocol:
<D> (uchar) b0 .. (uchar) b7
