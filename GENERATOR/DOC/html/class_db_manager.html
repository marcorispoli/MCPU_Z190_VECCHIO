Detailed Description

This class implements the SQL access to the Generator Error Messages database.

The Sedecal generator provides a SQLLite Database containing the Generator System Error descriptions.

The database is stored into C:\OEM\AppData\system_messags.sqlite.

The Class allows to open an arbitrary sqlite database with an arbitrary path in order to be more flexible for future path and name
modifications.
