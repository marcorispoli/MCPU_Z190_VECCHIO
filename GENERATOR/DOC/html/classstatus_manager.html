Detailed Description

This is the statusManager implementation class.

This class implements a Status Machine to control the Generator workflow.

The basic status that the class handle are:

  • SMS_SMART_HUB_CONNECTION: this is the status controlling the connection with the smart Hub
  • SMS_WAIT_GENERATOR_CONNECTION, SMS_SETUP_GENERATOR, SMS_SERVICE, SMS_IDLE, SMS_CLEAR_SYS_MESSAGES,
