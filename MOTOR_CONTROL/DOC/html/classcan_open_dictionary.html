◆ formatVal()

uint canOpenDictionary::formatVal ( uint  val)

Format a value size based on the OD type.

This function format the parameter value to the size of the OD.

The parameter is an unsigned long value. It is truncated to 1 Byte/2Byte/3Byte or 4 Byte
based on the actual format of the Object Dictionary.

USAGE

When it is needed to make a comparison from an arbitrary value and a content of
an Object register, before to make the comparison, the arbitrary value shall be
formatted to be compliance with the size of the Register value.
For example, if the Object register is a 2 byte data size,
before o be compared with an arbitrary uint variable, the vaRIABLE SHALL BE FORMATTED:

uint variable = ... someting
OD.getVal() == variable --> WRONG COMPARISON

uint variable = ... someting
OD.getVal() == OD.formatVal(variable) --> CORRECT COMPARISON

Parameters
valthis is the value that needs to be formatted
Returns
formatted result value
