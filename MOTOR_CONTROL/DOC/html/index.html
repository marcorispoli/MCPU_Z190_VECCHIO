GENERAL OVERVIEW

The Application can run without debug messages or alternatively can be launched with -file, -win or -console option:

  • -win: allows to run the application with a graphical window to provide manual interaction with the Sedecal Generator;
  • -file: the Application redirects the debug messages to a file in C:/OEM/Logs/GANTRY/Motors.log
  • -console: redirects the debug messages to the standard console

APPLICATION MODULES

The Application implements the following modules:

  • Application Module. : defines constant along the application
  • Library Modules Used in the Application. : this is the set of the libraries used in the current application;
  • Application Command Interface Module. : this is the communication socket to receive command from Gantry;
  • Nanotec Devices Implementation module : this is the module implementing the Can Open Protocol of the Nanotec PD4 Motors;
  • TRX communication protocol module. : this is the module implementing the TRX motor control;
  • armModule : this is the module implementing the ARM motor control;
  • liftModule : this is the module implementing the LIFT motor control;
  • slideModule : this is the module implementing the SLIDE motor control;
  • bodyModule : this is the module implementing the BODY motor control;
  • Debug Window Module : this is an optional Windows interface used for Service/Debug;
