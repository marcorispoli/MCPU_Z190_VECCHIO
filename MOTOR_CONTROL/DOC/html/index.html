APPLICATION MODULES

The Application implements the following modules:

  • Application Module. : defines constant along the application
  • interfaceModule : this is the communication socket to receive command from Gantry;
  • Can Driver Communication Module. : this is the communication socket to connect the Can Driver application;
  • CanOpen Protocol implementation. : this is the module implementing the Can Open Protocol;
  • TRX communication protocol module. : this is the module implementing the TRX motor control;
  • armModule : this is the module implementing the ARM motor control;
  • liftModule : this is the module implementing the LIFT motor control;
  • slideModule : this is the module implementing the SLIDE motor control;
  • bodyModule : this is the module implementing the BODY motor control;
  • windowModule : this is an optional Windows interface used for Service/Debug;
