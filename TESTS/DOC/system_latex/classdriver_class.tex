\doxysubsection{Detailed Description}
This class implements the DUT class can\+Driver\+Interface. 

This is the driver class implementation test. The class inherit the can\+Driver\+Interface that is the class under test.

The class implements the hardware interface methods that a can driver should implement\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{classdriver_class_a861d4b80e6e45e0aea90c9e7b48e694f}{hardware\+Init()}}\+: shall be implemented in order to initi the specific hardware inetrface;
\item \mbox{\hyperlink{classdriver_class_ae4c1b26c92bfa7ea4845480eb54013ac}{hardware\+Shutdown()}}\+: shall be implemented in order to terminates the hardware activities;
\item \mbox{\hyperlink{classdriver_class_a55b6324c7e8eba48147852839a9707aa}{open\+Port()}}\+: this shall open a given reception filter;
\item \mbox{\hyperlink{classdriver_class_ab00bbd659f50f9e4520c49402040c04d}{close\+Ports()}}\+: closes all the reception filters;
\item \mbox{\hyperlink{classdriver_class_a9a5a76ef7b7ad56eb1444948c5297aff}{run()}}\+: activates the bus arbitration;
\item \mbox{\hyperlink{classdriver_class_a7314d6b7d7306424f0df947d0b690039}{pause()}}\+: stops the bus arbitration;
\item \mbox{\hyperlink{classdriver_class_a8d9ec8f80aff87230eff2f2af1f100d7}{driver\+Tx\+Rx\+Data()}}\+: implements the simulated tx/rx data exchange
\end{DoxyItemize}

The Tx\+Rx simulated received frame (rxframe) implementation are\+:


\begin{DoxyItemize}
\item txframe.\+can\+Id == 1 -\/\texorpdfstring{$>$}{>} rxframe.\+data\mbox{[}0\mbox{]} = hardware initialization status (1 or 0);
\item txframe.\+can\+Id == 2 -\/\texorpdfstring{$>$}{>} rxframe.\+data\mbox{[}0\mbox{]}\mbox{[}1\mbox{]} = FILTER\+\_\+\+ID (LSB first);
\item txframe.\+can\+Id == 3 -\/\texorpdfstring{$>$}{>} rxframe.\+data\mbox{[}0\mbox{]}\mbox{[}1\mbox{]} = FILTER\+\_\+\+MASK (LSB first);
\item txframe.\+can\+Id == 4 -\/\texorpdfstring{$>$}{>} rxframe.\+data\mbox{[}0\mbox{]} = Extended mode status;
\item txframe.\+can\+Id == 5 -\/\texorpdfstring{$>$}{>} rxframe.\+data\mbox{[}0\mbox{]} = Running mode status;
\item txframe.\+can\+Id == 6 -\/\texorpdfstring{$>$}{>} rxframe.\+data\mbox{[}0\mbox{]}\mbox{[}1\mbox{]} = TIMEOUT (LSB first);
