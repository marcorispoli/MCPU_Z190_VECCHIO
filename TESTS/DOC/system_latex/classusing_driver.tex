\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classusing_driver}{using\+Driver}} class. 

This class implements a test class using the driver under test.

The class implements the necessary interface methods\+:
\begin{DoxyItemize}
\item signal \mbox{\hyperlink{classusing_driver_a8da9cd2362c8fdda3b30ffc2fa5a58dd}{tx\+To\+Driver()}}\+: this slot is connected with the Driver reception slot;
\item public \mbox{\hyperlink{classusing_driver_aaae6472aa546d90f38417df65a866d1c}{tx\+Data()}}\+: method to be used to send data\+: internally it triggers the \mbox{\hyperlink{classusing_driver_a8da9cd2362c8fdda3b30ffc2fa5a58dd}{tx\+To\+Driver()}} signal;
\item public slot \mbox{\hyperlink{classusing_driver_a54caebebd28e7b205479d7763933dad5}{device\+Rx\+Slot()}}\+: required to receive the reception data from the driver
\end{DoxyItemize}

In order to let the test function to wait for Queued data exchange the class implements an extra signal\+:
\begin{DoxyItemize}
\item signal \mbox{\hyperlink{classusing_driver_a10e43cc92e71e28a794dfb80a83f66cc}{rx\+Done()}}\+: this signal is emitted as soon as the \mbox{\hyperlink{classusing_driver_a54caebebd28e7b205479d7763933dad5}{device\+Rx\+Slot()}} is called.
\end{DoxyItemize}

In the class datata the received frame and od\+Index is stored into the rxframe and rx\+Index variables.
